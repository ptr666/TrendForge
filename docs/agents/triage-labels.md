# Triage Labels

技能使用五个标准 triage 角色。本文把这些角色映射到本仓库本地 markdown issue tracker 使用的实际 label 字符串。

| mattpocock/skills 中的 label | 本仓库 tracker label | 含义 |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | 需要维护者评估 |
| `needs-info` | `needs-info` | 等待报告者补充信息 |
| `ready-for-agent` | `ready-for-agent` | 已完整说明，可交给 AFK agent |
| `ready-for-human` | `ready-for-human` | 需要人类实现 |
| `wontfix` | `wontfix` | 不会处理 |

当技能提到某个角色时，使用表格中对应的 label 字符串。
