# 对话式公众号一键发布模板（升级版）

把下面任意一种模板直接发给我就行。

---

## 模板 A：最简可发版

适合你只想快速起稿、快速进草稿箱。

```text
请直接生成并发布公众号草稿：
主题：【你的主题】
目标读者：【你的目标读者】
传播目标：【你希望读者看完后产生什么行动 / 认知】
参考资料：【可选，没有就写“无”】
文章风格：【可选，例如：真诚、锋利、冷静、实战】
是否使用本地封面图：【可选，填路径；没有就写“无”】
```

### 最简示例

```text
请直接生成并发布公众号草稿：
主题：普通人怎么用 AI 建立第二收入
目标读者：对 AI 感兴趣但没有技术背景的上班族
传播目标：让读者相信 AI 不是只能看热闹，而是可以立刻开始做具体项目
参考资料：无
文章风格：真诚、实战
是否使用本地封面图：无
```

## 模板 B：增强成稿版

适合你希望文章**更像公众号成品文**，而不是只给一个普通说明文初稿。

```text
请直接生成并发布公众号草稿：
主题：【你的主题】
目标读者：【你的目标读者】
传播目标：【你希望读者看完后产生什么行动 / 认知】
核心观点：【这篇文章最想打透的一句话】
文章风格：【例如：真诚、锋利、冷静、克制、实战、故事化】
开头感觉：【例如：代入感强 / 直接切痛点 / 先讲反常识 / 像朋友聊天】
结构倾向：【例如：3 段式 / 问题-分析-建议 / 先破题再拆解】
希望包含的案例或场景：【可选】
希望给读者的行动建议：【可选】
参考资料：【可选，没有就写“无”】
是否使用本地封面图：【可选，填路径；没有就写“无”】
```

### 增强示例

```text
请直接生成并发布公众号草稿：
主题：为什么普通人做 AI，别一开始就盯着爆款和变现
目标读者：想做 AI 副业、但总在观望和收藏资料的上班族
传播目标：让读者意识到，先跑通一条小闭环，比研究所有机会更重要
核心观点：普通人做 AI，真正拉开差距的不是信息量，而是先把一条可重复执行的链路做出来
文章风格：真诚、克制、实战
开头感觉：像朋友聊天，但第一段就说到读者痛点
结构倾向：先破题，再拆常见误区，最后给行动建议
希望包含的案例或场景：总在收藏提示词、看博主拆项目，但迟迟没有发出第一条内容的人
希望给读者的行动建议：给出 3 个今天就能开始的小动作
参考资料：无
是否使用本地封面图：无
```

## 如果你想让我直接按“升级骨架”来写

你可以额外补一句：

```text
请把这篇文章写得更像公众号成品文：
- 开头要有导语感
- 中间要有一句可传播观点
- 每一节不要只像说明文，要更像在和读者说话
- 结尾要有自然收束和行动引导
```

---

## 我会自动完成的事情

1. 根据主题 / 目标读者 / 传播目标生成文章结构
2. 自动补成更适合公众号的导语、观点句、分节展开和收尾
3. 生成公众号 Markdown 正文
4. 写入 `articles/` 目录
5. 更新 `config.json` 中的当前文章路径
6. 如果提供本地封面图，则优先使用本地图
7. 如果没有本地封面图，则走 AI 封面图或兜底封面策略
8. 自动发布到公众号草稿箱

---

## 你如果想进一步控稿，可以直接补这些要求

这些要求不一定每次都要写，但你一写，我就会按这个方向收：

- **导语感**：开头更像公众号，不像说明文
- **传播句**：中间要有一句能单独摘出来传播的话
- **案例感**：多用真实场景，不只讲概念
- **行动感**：结尾给明确建议，不空收
- **克制度**：不要太鸡血，不要太像营销号
- **锋利度**：观点更鲜明一点，不要太平

示例：

```text
这篇请收得更像公众号成稿：
- 开头要强一点
- 中间给我一句能当金句摘出来的话
- 不要太像教程文档
- 结尾给读者一个明确行动建议
```
